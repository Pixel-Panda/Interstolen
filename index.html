Projects

SnipBin

SnipBin is a small, open-source, lightweight, elegant and blazing fast website for pasting code. It can support almost all programming languages. After seeing that PasteBin was the only know options to paste code, I wanted to create something of my own. PasteBin was absolutely littered with Ads, was freemium, and was very unpleasant to look at, so I made this! It has basic functionality, doesn't litter your screen with ads, and doesn't burn out your eyes when you look at it.

FastBot

FastBot is a cool little Discord bot with dozens of different commands. It was built with the thought of, "How can I improve the user's experience in this server overall, to make them reccomend others?", over a period of weeks. This bot has options to customize EVERYTHING about it, and is open-source. It covers the area of jokes and silly fun, but also has features like ghost ping detection, and spam protection.

SketchMessage

SketchMessage is a online chatting app, the only twist is, you draw to communicate! This was made by me as a side project, and I wanted to make something using Socket... but I didn't want it to be just any other chat app, so, I gave this one a twist, which leads to some pretty cool (and sometimes frantic) games full of people scribbling their brains out :P

AwesomeSource (in development)

This is my latest project I've been working on! Looking for open-source projects to contribute to can be a hassle, and AwesomeSource tries to fix that problem! Here, maintainers and project owners can post their project, the tech it uses, and other required details for users to make the search to find open source projects much easier. Another use of making this was to get my feet dirtier with using TypeScript, as I've heard a lot of people speaking very highly about it.

Numifier

This was my first every Python project, and I'm happy with it. It's a small command line tool which you can use it to turn regular text in the command line, into numbers. As an example, if you type "Hi there", it would output "H1 7H3R3". It doesn't really have much of a goal/motive to it, but it gave me some minimal experience in Python's syntax, and was another one of my silly projects.

Elanchau

Elanchau was a small game that I made for the Game Maker's Toolkit (GMTK) jam 2020, using the Unity game engine and C#. The theme of the jam was "out of control", so I came up with an idea of a player being out of control, shooting bullets which bounce everywhere and cause chaos! Here, you try to both kill the enemies to increase your score, and avoid being killed in the process, due to the very limited room you have. Note: this is the only project I have made which is not open-source, the files were lost before I could make them public.

The Basket

Another game made for another jam, this time, it was for the Extra Credits game jam. The theme for this one was "Take care", so I made a game, where a family has gone on a picnic and a team of 120 birds are attacking you, and you need to somehow stop them! The main purpose for this was to improve my art skills, as the art in this game was from a vector/Kurzgesagt inspired artstyle, which was fun to try out, and turned out quite nicely, in my opinion.

All About the Jelly

All about the jelly is a experimental game, about trying to simulate jelly in the Unity game engine. It a range of customization and recoil options, to give the user a more satisfying and fun jelly playing experience.
